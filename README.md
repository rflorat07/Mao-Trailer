# APP - Mao Trailer  ( IOS 11 - Swift 4 )

## Features

* User Authentication - create account/login
* Featured and Latest Movies
* Featured and Latest TV Shows
* Search and find movies from API (TMDb)
* Save Favorite movie to firebase backend
